this is a proxy server for processing images with replicate from an ios app called "Colorise". Colorise is a app that allows you to colorise black and white images.

The app sends the image to the server, the server processes the image with replicate and returns the processed image to the app.

The server is hosted on railway.app and the app is hosted on the app store.

The server is written in typescript using the hono framework.
